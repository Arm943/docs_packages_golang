Методы поиска подстроки в строке

1. strings.Contains(s, substr string) bool

Что делает?
Проверяет, содержит ли строка s подстроку substr.

Применение:

  • Проверить, есть ли слово "Go" в "I love Go!".
  • Проверить, есть ли имя "Армен" в "Привет, Армен!".

Пример кода:


package main
import (
    "fmt"
    "strings"
)

func main() {
    fmt.Println(strings.Contains("I love Go!", "Go"))  // true
    fmt.Println(strings.Contains("Hello, world!", "Hi")) // false
}

2. strings.ContainsAny(s, chars string) bool

Что делает?
Проверяет, содержит ли строка s хотя бы один символ из набора chars.

Применение:

  • Проверить, есть ли в строке хотя бы одна буква из "abc".
  • Проверить, содержит ли пароль запрещённые символы "$#%&".

Пример кода:


package main
import (
    "fmt"
    "strings"
)

func main() {
    fmt.Println(strings.ContainsAny("hello", "aeiou"))  // true (есть 'e' и 'o')
    fmt.Println(strings.ContainsAny("hello", "xyz"))    // false (нет 'x', 'y', 'z')
    fmt.Println(strings.ContainsAny("password123!", "!$%")) // true (есть '!')
}

3. strings.ContainsRune(s string, r rune) bool

Что делает?
Проверяет, содержит ли строка s конкретную руну (символ).

Применение:

  • Проверить, есть ли буква 'Г' в "Голанг".
  • Проверить, встречается ли '€' в тексте.

Пример кода:


package main
import (
    "fmt"
    "strings"
)

func main() {
    fmt.Println(strings.ContainsRune("Голанг", 'Г'))  // true
    fmt.Println(strings.ContainsRune("Hello", 'H'))   // true
    fmt.Println(strings.ContainsRune("123456", '7'))  // false
}

4. strings.HasPrefix(s, prefix string) bool

Что делает?
Проверяет, начинается ли строка s с подстроки prefix.

Применение:

  • Проверить, начинается ли URL с "https://".
  • Проверить, начинается ли имя файла с "log_".

Пример кода:


package main
import (
    "fmt"
    "strings"
)

func main() {
    fmt.Println(strings.HasPrefix("https://google.com", "https://"))  // true
    fmt.Println(strings.HasPrefix("file.txt", "log_"))               // false
}

5. strings.HasSuffix(s, suffix string) bool

Что делает?
Проверяет, заканчивается ли строка s подстрокой suffix.

Применение:

  • Проверить, заканчивается ли файл на ".jpg".
  • Проверить, заканчивается ли предложение точкой ".".

Пример кода:


package main
import (
    "fmt"
    "strings"
)

func main() {
    fmt.Println(strings.HasSuffix("document.pdf", ".pdf"))  // true
    fmt.Println(strings.HasSuffix("image.jpeg", ".png"))    // false
}
